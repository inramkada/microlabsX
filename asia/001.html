NODE 001

This node exposes a live NVIDIA Omniverse viewport streamed directly to the browser. The scene runs remotely on a GPU server and is delivered using WebRTC.

This is not a video. The viewport is interactive and rendered in real time using RTX.

Node 001 demonstrates how Omniverse can be integrated as a web-native tool while preserving USD workflows, high-end rendering and real-time interaction.
