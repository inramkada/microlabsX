NODE 001

This node exposes a live NVIDIA Omniverse viewport streamed to the browser. The goal is to demonstrate real-time collaboration, RTX rendering, and USD-based workflows directly inside the web.

This is not a video. The scene runs remotely on a GPU server and is streamed interactively using WebRTC.
